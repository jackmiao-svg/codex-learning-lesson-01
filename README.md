# 个人介绍网页

## 项目简介

这是一个用于学习 Vibe Coding 的简单个人介绍网页，适合编程初学者练习 HTML、CSS 和 JavaScript。

## 当前功能

- 显示网页标题和个人介绍
- 显示“开始学习”按钮
- 点击按钮后显示鼓励文字
- 每次点击按钮，小目标完成数量会增加 1

## 使用的技术

- HTML
- CSS
- JavaScript

## 如何在本地打开

在项目文件夹中找到 `index.html`，然后用浏览器打开即可查看网页。

## 项目文件结构

```text
lesson-01/
├── index.html
├── style.css
├── script.js
└── README.md
```

## 学习目标

- 理解 HTML 负责网页内容结构
- 理解 CSS 负责网页样式
- 理解 JavaScript 负责网页交互
- 学习使用 Git 管理项目版本
